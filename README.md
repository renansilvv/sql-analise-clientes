## Análise de Vendas com SQL

Este projeto foi desenvolvido com o objetivo de praticar análise de dados utilizando SQL e Power BI a partir de um dataset real de vendas.

A ideia principal é explorar os dados e extrair informações relevantes sobre desempenho de pedidos, clientes, localização, categorias de produtos e valores de vendas.

## Sobre os dados

Os dados utilizados neste projeto são públicos e representam registros de vendas.

Fonte:
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting

## Análises realizadas

As consultas foram desenvolvidas com foco em responder perguntas básicas de negócio, como:

- volume total de vendas
- vendas por região
- vendas por categoria
- produtos com maior receita
- clientes com maior volume de compras
- evolução das vendas ao longo do tempo

## Power BI

Foi desenvolvido um dashboard utilizando Power BI com foco em:

- visão geral de vendas
- desempenho por produto
- distribuição geográfica
- análise temporal

## Insights

- a região West concentra a maior parte das vendas, indicando maior demanda ou presença de mercado.
- a categoria Technology apresenta maior ticket médio em comparação com as demais.
- alguns clientes concentram grande volume de compras, sugerindo oportunidades de retenção.
- as vendas apresentam variação ao longo do tempo, com picos em determinados períodos.
